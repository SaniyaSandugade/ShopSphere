# ShopSphere

Modern Full Stack E-Commerce Web Application

---

## Features

### User Features

- Product Listing
- Product Details
- Add To Cart
- Checkout System
- Payment Method Selection
- Orders Page

### Admin Features

- Admin Dashboard
- Product Management
- Orders Management
- Revenue Analytics
- Sales Charts

### Security Features

- JWT Authentication
- Helmet Security
- Rate Limiting

### Notifications

- Email Notifications using Nodemailer

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ShopSphere.git
```

---

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

---

### Backend Setup

```bash
cd backend

npm install

node server.js
```

---

## Environment Variables

Create `.env` file inside backend folder.

```env
MONGO_URL=your_mongodb_url

JWT_SECRET=your_secret

EMAIL_USER=your_email

EMAIL_PASS=your_app_password

CLOUD_NAME=your_cloudinary_name

API_KEY=your_cloudinary_key

API_SECRET=your_cloudinary_secret
```
